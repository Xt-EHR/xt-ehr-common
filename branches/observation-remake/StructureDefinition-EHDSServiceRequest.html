Logical Model: Service request model

Official URL: https://www.xt-ehr.eu/specifications/fhir/StructureDefinition/EHDSServiceRequest Version: 0.1.0
Draft as of 2025-07-08 Computable Name: EHDSServiceRequest

EHDS refined base model for Specification of requested service or services

Usages:

  • Use this Logical Model: Imaging report body model, Laboratory order body model, Laboratory report body model and Observation model
  • Refer to this Logical Model: Encounter model and Imaging study model
