Logical Model: DataSet model

Official URL: https://www.xt-ehr.eu/specifications/fhir/StructureDefinition/EHDSDataSet Version: 0.1.0
Draft as of 2025-05-20 Computable Name: EHDSDataSet

Common elements (including header) for all documents and their independently functioning parts, e.g FHIR resources.

Usage:

  • Derived from this Logical Model Profile: Medication dispense decline model, Document model, Medication dispense model and Medication prescription model
