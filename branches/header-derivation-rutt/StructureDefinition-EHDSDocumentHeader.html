Logical Model: Document header model

Official URL: https://www.xt-ehr.eu/specifications/fhir/StructureDefinition/EHDSDocumentHeader Version: 0.1.0
Draft as of 2025-04-16 Computable Name: EHDSDocumentHeader

EHDS refined base model for document header data elements, including the common header for all resources. Data relevant to document type and its content for administrative and searching purposes.

Usage:

  • Derived from this Logical Model Profile: Laboratory report header model
