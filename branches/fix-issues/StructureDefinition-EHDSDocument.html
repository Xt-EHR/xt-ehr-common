Logical Model: Document model

Official URL: https://www.xt-ehr.eu/specifications/fhir/StructureDefinition/EHDSDocument Version: 0.1.0
Draft as of 2025-05-26 Computable Name: EHDSDocument

EHDS refined base model for common document data elements, including the common header. Data relevant to document type and its content for administrative and searching purposes.

Usage:

  • Derived from this Logical Model Profile: Laboratory report header model and Patient summary model
