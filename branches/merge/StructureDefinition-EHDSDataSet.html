Logical Model: DataSet model

Official URL: https://www.xt-ehr.eu/specifications/fhir/StructureDefinition/EHDSDataSet Version: 0.2.0
Draft as of 2025-10-08 Computable Name: EHDSDataSet

Common elements (including header) for all documents and their independently functioning parts, e.g FHIR resources.

Usages:

  • Derived from this Logical Model: Advance directive model, Alert model, Allergy intolerance model, Care plan model... Show 23 more, Condition model, Current pregnancy status model, Device use model, Medication dispense decline model, Document model, Encounter model, Episode of care model, Family member history model, Functional status, Imaging study model, Immunisation model, Infectious contact model, Medication administration model, Medication dispense model, Medication prescription model, Medication statement model, Observation model, Pregnancy history model, Procedure model, Service request model, Social history model, Substance use model and Travel history model
