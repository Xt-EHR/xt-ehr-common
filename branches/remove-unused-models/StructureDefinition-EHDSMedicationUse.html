Logical Model: Medication use model

Official URL: http://www.xt-ehr.eu/fhir/models/StructureDefinition/EHDSMedicationUse Version: 0.2.1
Draft as of 2026-02-05 Computable Name: EHDSMedicationUse

Statement about a single medication as part of a medication summary.

Usages:

  • Use this Logical Model: Discharge Report model and Patient summary model
