Logical Model: Report header model

Official URL: https://www.xt-ehr.eu/specifications/fhir/StructureDefinition/EHDSReportHeader Version: 0.1.0
Draft as of 2025-07-03 Computable Name: EHDSReportHeader

EHDS refined base model for Report header data elements. Common for hospital discharge, laboratory and medical imaging reports.

Usages:

  • This Logical Model is not used by any profiles in this Implementation Guide
